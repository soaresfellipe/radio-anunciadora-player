Adicione o player da Rádio Anunciadora aos sites da Arquidiocese de Niterói.

== Descrição ==
O plugin Rádio Anunciadora Player permite adicionar o player da Rádio Anunciadora aos sites da Arquidiocese de Niterói.

== Instalação ==
1. Faça o upload da pasta `radio-anunciadora-player` para o diretório `wp-content/plugins/`.
2. Ative o plugin no menu "Plugins" do WordPress.

== Uso ==
Use o shortcode `[radio_anunciadora_player]` para exibir o player da Rádio Anunciadora em qualquer página ou post.

== Estilização ==
Você pode personalizar a aparência do player da Rádio Anunciadora editando o arquivo CSS `radio-anunciadora-player.css` localizado na pasta do plugin.
